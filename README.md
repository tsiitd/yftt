# TrendTop

A free, public, mobile-first website surfacing **Yahoo Finance trending and most-active stocks that are within 1% of their 52-week high** — potential breakout candidates at a glance.

**Live site:** https://tsiitd.github.io/Yf_TrendTop/

Data is refreshed automatically via GitHub Actions every 15 minutes during market hours (weekdays, 02:00–22:00 UTC). No login, no ads, no tracking.

For architecture details see [`ai/context.md`](ai/context.md).
